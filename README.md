# blog-api-nodejs


REST API для блогової платформи з повноцінною системою автентифікації, CRUD операціями для постів та коментарів. Розроблено в рамках дисципліни "Розробка програмного забезпечення на Node.js".


##  Особливості

-  **JWT автентифікація** з захищеними маршрутами
-  **Ролева система** (user/admin)
-  **Повний CRUD** для постів та коментарів
-  **Пошук та фільтрація** з пагінацією
-  **Валідація даних** на всіх рівнях
-  **Повне тестування** (Unit, Integration, E2E)
-  **CI/CD** з GitHub Actions
-  **Моніторинг** та логування
